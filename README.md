# Life in Numbers

Life in Numbers is a cinematic predictive lifestyle analytics platform that transforms everyday habits into visual long-term life projections.

The experience combines futuristic interface design, behavioral simulations, biological age estimation, circadian rhythm analysis, and interactive dashboards to help users understand how small daily routines compound over time.

Built with a strong focus on immersive UI/UX, the platform blends data visualization with storytelling to create an engaging self-analysis experience.

## Features

- Interactive lifestyle analytics cockpit
- Biological age & vitality estimation
- 24-hour circadian rhythm simulator
- Future self projection system
- Dynamic charts and data visualization
- Responsive cinematic interface
- Real-time parameter interaction
- Habit-based long-term lifestyle modeling

## Installation

Clone the repository:

```bash
git clone https://github.com/arpiitaaa/Life-in-Numbers.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
